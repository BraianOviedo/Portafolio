OdinAI

Preview

Este proyecto es una app web desarrollada en Angular que brinda el servicio de un chatbot basado en uno de los LLMs provistos por Groq. Permite realizar cualquier tipo de consulta en forma de texto y guardar las respuestas en colecciones que el mismo usuario puede crear. También se incorporan otras funcionalidades como reconocimiento visual para manejar las colecciones de una forma mas interactiva e innovadora.
